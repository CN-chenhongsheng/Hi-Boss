# TabBar 图标说明

请在此目录下添加以下图标文件（推荐尺寸 81x81 像素，PNG 格式）：

## 所需图标

| 文件名 | 说明 |
|--------|------|
| home.png | 首页图标（未选中） |
| home-active.png | 首页图标（选中） |
| chart.png | 统计图标（未选中） |
| chart-active.png | 统计图标（选中） |
| user.png | 我的图标（未选中） |
| user-active.png | 我的图标（选中） |

## 图标颜色建议

- 未选中状态：#999999
- 选中状态：#5D87FF

## 获取图标

推荐从以下网站获取图标：

1. [阿里巴巴矢量图标库](https://www.iconfont.cn/)
2. [Flaticon](https://www.flaticon.com/)
3. [Icons8](https://icons8.com/)

下载后请确保：
- 格式为 PNG
- 尺寸为 81x81 像素
- 背景透明
